# HTML

_HTML/CSS_&#x20;

어떠한 언어와도 사용될 수 있는 범용성

ex) 포토샵을 한글과 연동시킬 수 있는 방법은 거의 없지만 html/css는 거의 모든 언어에서 사용할 수 있음



```css
<!DOCTYPE html> 
<HTML>
</HTML> 
```

&#x20;HTML 태그이다 라는 의미



`<head>`&#x20;

여행가방의  수화물 태그같은 역할



```html
<meta charset="UTF-8" />
```

한글이 정상적으로 출력



```css
overflow: hidden;
```

그다음에 오는 HTML요소들이 float의 영향을 받지 않도록 함
